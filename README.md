# ATM Application in Python

A desktop-based ATM simulation application developed using Python. The application provides a graphical user interface (GUI) that simulates basic ATM operations such as account access, balance inquiry, cash withdrawal, cash deposit, and transaction handling.

---

## Features

- User-friendly graphical interface
- Secure PIN-based login
- Balance inquiry
- Cash withdrawal
- Cash deposit
- Transaction validation
- Simple and intuitive interface
- Python-based implementation

---

## Technologies Used

- Python 3
- Tkinter (GUI)

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Thiru2006/ATM-Application-in-Python.git
```

### Navigate to the project

```bash
cd ATM-Application-in-Python
```

### Run the application

```bash
python ATM_GUI.py
```
